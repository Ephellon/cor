cor

Cor is a language that compiles to javascript, thinked to develop large and maintainable client-side applications.

Get Started

Designed for productivity

Cor makes client-side development simple and nice. It is made to bring structure, reusability and scalability of maintenance to the front-end.

Programming structures

Cor easily lets you to make large front-end applications by providing a clear syntax, classes, modules and packages to keep organized code. You can write it as reusable as you want.

Development experience

Cor asynchronously load and compiles sources, allowing easy debugging in the browser through source-maps. It ships CLI tools to build your application once it is ready for delivery.

Embracing the web

With Cor you can utilize your favorite javascript library thanks to the Cor loader. It can be extended to support preprocessors such as LESS, Handlebars, Mustache, Showdown, and every else.
